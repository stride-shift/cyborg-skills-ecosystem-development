# 52 Cyborg Skills Catalog

**Complete catalog of skills for the weekly video series**

## Category 1: Content Transformation (Skills 1-10)

### Skill 1: Meeting to Summary
**Habit Connection:** Explain It, Improve It
**Description:** Transform any meeting transcript into a structured, actionable summary in under 60 seconds. Extract decisions made, action items with owners, open questions, and next steps.
**Key Prompt:** "Summarize this meeting with: key decisions, action items with owners, open questions, and next steps"
**Use Cases:** Post-meeting documentation, stakeholder updates, personal reference
**Difficulty:** Beginner

### Skill 2: Meeting to Execution
**Habit Connection:** Plan It, Improve It
**Description:** Go beyond summarization to create actual deliverables from meeting content. Turn discussions into proposals, prototypes, or project plans.
**Key Prompt:** "Based on this meeting, create a draft [proposal/project plan/brief] that addresses the key points discussed"
**Use Cases:** Project kickoffs, client meetings, strategy sessions
**Difficulty:** Intermediate

### Skill 3: Blog to Infographic
**Habit Connection:** Explain It, Improve It
**Description:** Convert written blog content into visual infographic format, extracting the key points and structuring them for visual consumption.
**Key Prompt:** "Extract the 5-7 key points from this blog post and structure them for an infographic with a clear visual hierarchy"
**Use Cases:** Content marketing, social media, presentations
**Difficulty:** Beginner

### Skill 4: Infographic to Social
**Habit Connection:** Improve It
**Description:** Transform infographic content into social media posts optimized for different platforms (LinkedIn, Twitter, etc.).
**Key Prompt:** "Create 3 LinkedIn posts from this infographic, each highlighting a different key point with a hook and call to engagement"
**Use Cases:** Social media marketing, content repurposing
**Difficulty:** Beginner

### Skill 5: Email Thread to Deck
**Habit Connection:** Plan It, Explain It
**Description:** Convert long email threads into presentation slides that capture the key discussions, decisions, and next steps.
**Key Prompt:** "Turn this email thread into a 5-slide summary deck: context, key points discussed, decisions made, open items, next steps"
**Use Cases:** Executive briefings, project updates, stakeholder communication
**Difficulty:** Intermediate

### Skill 6: Research to Brief
**Habit Connection:** Explain It, Improve It
**Description:** Condense extensive research into executive-friendly briefings that highlight key findings and recommendations.
**Key Prompt:** "Condense this research into a 1-page executive brief with: key findings, implications, and recommended actions"
**Use Cases:** Strategic planning, executive communication, decision support
**Difficulty:** Intermediate

### Skill 7: Long-form to Bullets
**Habit Connection:** Explain It
**Description:** Extract key points from lengthy documents into scannable bullet-point format while preserving essential information.
**Key Prompt:** "Extract the essential points from this document as bullet points, organized by theme, maximum 20 bullets"
**Use Cases:** Document review, meeting prep, quick reference
**Difficulty:** Beginner

### Skill 8: Conversation to FAQ
**Habit Connection:** Explain It, Guide Me
**Description:** Mine conversations, support tickets, or discussions for frequently asked questions and create comprehensive FAQ content.
**Key Prompt:** "Identify the questions being asked in these conversations and create FAQ entries with clear, helpful answers"
**Use Cases:** Customer support, documentation, onboarding materials
**Difficulty:** Intermediate

### Skill 9: Feedback to Themes
**Habit Connection:** Critique It, Explain It
**Description:** Analyze feedback from surveys, reviews, or comments to identify patterns, themes, and actionable insights.
**Key Prompt:** "Analyze this feedback and identify: top 5 themes, sentiment for each, specific examples, and recommended actions"
**Use Cases:** Product development, customer experience, team improvement
**Difficulty:** Intermediate

### Skill 10: Data to Story
**Habit Connection:** Explain It, Imagine It
**Description:** Transform raw data into narrative insights that communicate findings in a compelling, memorable way.
**Key Prompt:** "Turn this data into a narrative that explains what happened, why it matters, and what we should do about it"
**Use Cases:** Reporting, presentations, stakeholder communication
**Difficulty:** Advanced

## Video Script Template

Each skill video follows this structure:

```
[0:00-0:10] HOOK
- Surprising capability or problem statement

[0:10-0:40] CONTEXT
- Why this skill matters
- When you'd use it

[0:40-3:40] DEMONSTRATION
- Show the skill in action
- 2-3 variations or refinements
- Common mistakes to avoid

[3:40-4:00] SUMMARY
- Key takeaway
- When to use this skill

[4:00-4:10] CTA
- Try it now prompt
- Next skill preview
```
