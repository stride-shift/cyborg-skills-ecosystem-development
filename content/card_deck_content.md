## PROMPT CARDS (49 Total)

### EXPLAIN IT Cards (E1-E7)

**E1**
```
"Explain this like I'm five"

Use when: Complex concept feels overwhelming
Result: Crystal-clear understanding
```

**E2**
```
"Explain this to someone in
a completely different field"

Use when: Need to communicate across domains
Result: Jargon-free explanation
```

**E3**
```
"Give me three different
analogies for this concept"

Use when: Struggling to grasp something abstract
Result: Multiple mental models to choose from
```

**E4**
```
"Break this into its
simplest components"

Use when: Facing something that feels too big
Result: Manageable pieces to work with
```

**E5**
```
"What's the 30-second
version of this?"

Use when: Need to communicate quickly
Result: Elevator-pitch clarity
```

**E6**
```
"Explain what I need to know
vs. what's nice to know"

Use when: Information overload
Result: Focused understanding
```

**E7**
```
"Explain this as if I'm
smart but unfamiliar"

Use when: Entering a new domain
Result: Respectful, clear onboarding
```
