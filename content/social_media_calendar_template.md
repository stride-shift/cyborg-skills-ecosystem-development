## Content Type Templates

### Type A: Framework Post
**Best for:** Monday slots
**Structure:**
```
[Hook - surprising claim or question]

Here's the framework:

1. [Point 1]
   → Brief explanation

2. [Point 2]
   → Brief explanation

3. [Point 3]
   → Brief explanation

[Key insight or takeaway]

[CTA - question or link]
```

**Character count:** 1,000-1,500

### Type B: Hot Take
**Best for:** Wednesday slots
**Structure:**
```
[Controversial opening statement]

[2-3 sentences explaining why]

[Evidence or example]

[What this means]

[Question to engage]
```

**Character count:** 500-800

### Type C: Story Post
**Best for:** Friday slots
**Structure:**
```
[Scene setting - 1-2 sentences]

[The problem or challenge]

[The turning point]

[The result]

[The lesson]

[Question or reflection]
```

**Character count:** 800-1,200

### Type D: List Post
**Best for:** Any day - high engagement
**Structure:**
```
[# things about X]

1. [Item]
2. [Item]
3. [Item]
4. [Item]
5. [Item]

[Which resonates most?]
```

**Character count:** 400-700

### Type E: Quote Card
**Best for:** Supplementary posts
**Components:**
- Short, impactful quote
- Attribution
- Branded visual template
- Caption with context (100-200 words)

## Quick Reference: Best Practices

### Optimal Posting Times (LinkedIn)
- **Best:** Tuesday-Thursday, 8-10 AM
- **Good:** Monday, Wednesday, Friday 8-9 AM
- **Avoid:** Weekends, late evenings

### Engagement Rules
- Respond to comments within 2 hours
- Ask follow-up questions
- Tag relevant people
- Never argue publicly
- Take heated discussions to DM

### Content Quality Checklist
- [ ] Hook in first line
- [ ] Value for reader clear
- [ ] No unexplained jargon
- [ ] Active voice
- [ ] Clear CTA or question
- [ ] Appropriate hashtags
- [ ] Proofread for errors
- [ ] On-brand tone
