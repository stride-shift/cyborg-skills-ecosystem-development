# Scream Machine - Content Extraction & Transformation Templates

**Ready-to-use prompts for the content creation pipeline**

## PART 1: EXTRACTION PROMPTS

### Template 1: Key Insights Extraction
```
Analyze this transcript and extract:

1. MAIN ARGUMENTS/THESES
   - Number each distinct argument
   - Include the core claim and why it matters

2. SUPPORTING EVIDENCE
   - Examples given for each argument
   - Data or stories that back up claims

3. NOVEL FRAMEWORKS OR MODELS
   - New ways of thinking introduced
   - Mental models or structures

4. CONTRARIAN OR SURPRISING CLAIMS
   - Statements that challenge conventional wisdom
   - Unexpected perspectives

5. PRACTICAL APPLICATIONS
   - What listeners should DO with this
   - Actionable takeaways

Format each clearly with headers and bullet points.
```

### Template 2: Quotable Moments Extraction
```
Find 15-20 quotable moments from this transcript that:
- Are self-contained (make sense without context)
- Express a strong point of view
- Would work as social media posts
- Are memorable and shareable
- Are 1-3 sentences maximum

For each quote provide:
- The exact quote (in quotation marks)
- Speaker (if identifiable)
- Category: [Hot Take / Framework / Prediction / Call to Action / Problem Statement]
- Brief context (1 sentence)
- Suggested hashtags

Prioritize quotes that are punchy, provocative, or profound.
```

### Template 3: Story Seeds Extraction
```
Identify potential stories and narratives in this transcript:

1. COMPLETE STORIES
   - Full narratives with beginning, middle, end
   - Include who, what, outcome

2. PARTIAL STORIES / ANECDOTES
   - References to examples that could be expanded
   - Incomplete mentions worth developing

3. ANALOGIES & METAPHORS
   - Comparisons used to explain concepts
   - Could be expanded into longer pieces

4. HYPOTHETICAL SCENARIOS
   - "What if" situations discussed
   - Future visions or possibilities

5. BEFORE/AFTER TRANSFORMATIONS
   - Changes described
   - Improvements or shifts

For each, note:
- The story seed
- Which habit it illustrates (if applicable)
- Potential uses (book, social, video)
- What additional detail would be needed
```

### Template 4: Framework Content Extraction
```
Extract all framework and concept explanations:

1. CONCEPT DEFINITIONS
   - Term: [exact term]
   - Definition: [how it's explained]
   - Why it matters: [significance]

2. EXPLANATIONS OF THE 7 HABITS
   - Which habit
   - How it's described
   - Examples given
   - Nuances or variations

3. PHILOSOPHICAL CONCEPTS
   - Extended Mind Hypothesis references
   - Transparent Equipment explanations
   - Cyborg identity discussions

4. ANALOGIES USED
   - The analogy
   - What it explains
   - How effective it is

5. DIFFERENTIATORS
   - What this ISN'T (course, training, etc.)
   - What makes it unique
   - Competitive positioning
```

## PART 2: TRANSFORMATION PROMPTS

### Template 5: Transcript to Blog Post
```
Transform this insight into a 600-word blog post:

INSIGHT: [paste insight]

Requirements:
- Hook in first sentence (surprising, provocative, or relatable)
- Explain the concept clearly in 2-3 paragraphs
- Include one concrete example or story
- End with actionable takeaway
- Tone: Confident but not preachy, slightly irreverent
- No jargon without explanation
- Short paragraphs (2-4 sentences)

Structure:
1. Hook (1 sentence)
2. Context/Problem (1 paragraph)
3. The Insight (2 paragraphs)
4. Example/Story (1-2 paragraphs)
5. So What/Action (1 paragraph)

Title should be intriguing, not clickbait.
```

### Template 6: Insight to LinkedIn Post
```
Create a LinkedIn post from this insight:

INSIGHT: [paste insight]

Requirements:
- Start with hook/claim (stops the scroll)
- 150-200 words maximum
- Use line breaks for readability
- End with question or call to engage
- Include 3-5 relevant hashtags

Format options (choose one):
A) Hot take + reasoning
B) Story + lesson
C) Framework breakdown (numbered list)
D) Myth busting

Tone: Professional but human, confident, conversational
```

### Template 7: Blog to Infographic Structure
```
Extract infographic content from this blog post:

BLOG POST: [paste post]

Create infographic structure with:

1. HEADLINE (5-7 words, impactful)

2. SUBHEAD (1 sentence context)

3. KEY POINTS (5-7 maximum)
   - Each point: Icon suggestion + 3-5 word label + 1 sentence explanation

4. VISUAL FLOW
   - Suggested layout (list, process, comparison, hub-spoke)
   - Visual hierarchy notes

5. CALL TO ACTION
   - What should viewer do next

6. SOURCE/ATTRIBUTION
```

### Template 8: Content to Social Media Suite
```
Create a social media content suite from this piece:

SOURCE CONTENT: [paste content]

Generate:

1. LINKEDIN POST (150-200 words)
   - Professional tone
   - Thought leadership angle

2. TWITTER/X THREAD (5-7 tweets)
   - Punchy, each tweet stands alone

4. ENGAGEMENT QUESTIONS (3 options)

```

### Template 9: Transcript to Video Script
```
Create a 5-minute video script from this content:

Structure:

- Stops viewer from scrolling
```

## PART 3: QUALITY PROMPTS

### Template 11: Voice & Tone Check
```
Review this content for voice and tone:

CONTENT: [paste content]

Check against these standards:
- Confident but not arrogant
- Slightly irreverent/quirky
- Accessible (no unexplained jargon)
- Active voice preferred
- Short sentences and paragraphs
- "Smart friend explaining" not "professor lecturing"

Identify:
1. Passages that don't match voice
2. Suggested revisions
3. Jargon that needs explanation
4. Sentences that could be shorter
```

### Template 12: Fact & Accuracy Check
```
Review this content for accuracy:

CONTENT: [paste content]

Check:
1. Claims that need verification
2. Statistics or data mentioned
3. Quotes - are they accurate?
4. Framework descriptions - correct?
5. Any potential misrepresentations

Flag anything that:
- Seems too strong/absolute
- Could be misinterpreted
- Needs a source
- Contradicts other content
```

### Template 13: Brand Consistency Check
```
Review for brand consistency:

CONTENT: [paste content]

Verify alignment with:

1. KEY MESSAGES
   - Behavior change, not learning
   - Transparent equipment
   - 7 habits framework
   - In-flow-of-work

2. POSITIONING
   - Not a course
   - Not AI training
   - Habit formation

3. TERMINOLOGY
   - Using correct terms
   - Consistent naming
   - Proper capitalization

4. TONE
   - Matches brand voice
   - Professional but accessible

Flag any inconsistencies and suggest corrections.
```
