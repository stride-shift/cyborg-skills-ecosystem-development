# Challenge Card Deck Specification (Physical Product)

## Purpose

Create a beautifully designed physical card deck that:
1. Serves as a tangible reminder of the 7 cyborg habits
2. Provides actionable prompts for daily AI use
3. Works as a standalone product or program companion
4. Functions as a marketing/brand touchpoint at events
5. Creates "environmental triggers" for habit formation

> "Even unopened boxes serve as brand reminders"

## Target Audience

- Cyborg Habits program participants (companion product)
- Event/expo attendees (giveaway/purchase)
- Professionals wanting physical AI productivity tool
- Gift-givers for colleagues interested in AI
- Teams wanting shared reference tool

## Product Design

### Card Specifications
- **Total Cards**: 56 cards
  - 7 habit divider cards
  - 7 cards per habit (49 prompt cards)
- **Size**: Standard poker size (2.5" x 3.5") or slightly larger
- **Material**: Premium card stock, matte or soft-touch finish
- **Corners**: Rounded for durability
- **Box**: Tuck box or magnetic closure box

### Visual Design

#### Design Aesthetic
- **Glass aesthetic** - Modern, clean, translucent feel
- **Consistent with brand** - Colors, typography from Cyborg Skills
- **Visual metaphors** - Each habit has an iconic visual
- **High-end feel** - Premium product, not cheap giveaway

## Card Content

### Habit 1: EXPLAIN IT (7 cards)
1. "Explain this like I'm five"
2. "Explain this to someone in [different field]"
3. "Give me three analogies for this concept"
4. "Break this down into its simplest components"
5. "What's the ELI5 version of this document?"
6. "Explain this using only common words"
7. "Summarize this for a busy executive in 30 seconds"

### Habit 2: PLAN IT (7 cards)
1. "Create a step-by-step plan for [goal]"
2. "Break this project into weekly milestones"
3. "What are the first three things I should do?"
4. "Map out the dependencies in this project"
5. "Create a decision tree for this situation"
6. "What's a realistic timeline for this?"
7. "Identify the critical path to completion"

### Habit 3: CRITIQUE IT (7 cards)
1. "Play devil's advocate on this idea"
2. "What are the top 5 risks I'm not seeing?"
3. "Poke holes in this plan"
4. "What would a skeptic say about this?"
5. "What assumptions am I making that could be wrong?"
6. "If this fails, what's the most likely reason?"
7. "What's the strongest argument against this?"

### Habit 4: GUIDE ME (7 cards)
1. "Walk me through how to [task] step by step"
2. "I've never done [X] before. Where do I start?"
3. "What would an expert do first in this situation?"
4. "Give me a checklist for [process]"
5. "What are the common mistakes beginners make here?"
6. "Guide me through this like a patient mentor"
7. "What do I need to know before attempting [X]?"

### Habit 5: IMAGINE IT (7 cards)
1. "What could this look like in 5 years?"
2. "Describe 3 different possible outcomes"
3. "If we had unlimited resources, what would we do?"
4. "Paint a picture of success for this project"
5. "What's the best-case scenario here?"
6. "Imagine we've already solved this. What did we do?"
7. "What possibilities am I not considering?"

### Habit 6: WHAT IF (7 cards)
1. "What if we did the opposite?"
2. "What if this assumption were false?"
3. "What if we had to do this in half the time?"
4. "What are the second-order effects of this decision?"
5. "What if our biggest competitor did this first?"
6. "What if we removed this constraint entirely?"
7. "What happens if this succeeds beyond expectations?"

### Habit 7: IMPROVE IT (7 cards)
1. "How can I make this 10% better?"
2. "What would the best version of this look like?"
3. "What's missing from this?"
4. "Suggest three ways to strengthen this argument"
5. "How would [expert/company] approach this differently?"
6. "What's the one change that would have the biggest impact?"
7. "Polish this to be presentation-ready"

## Production

### Design Process
1. Finalize card content and copy
2. Create visual design templates
3. Generate habit icons (Nano Banana)
4. Design card fronts and backs
5. Create divider cards
6. Design box/packaging
7. Prepare print-ready files

### Manufacturing Options

#### Option A: Print-on-Demand
- **Provider**: MakePlayingCards, PrinterStudio, or similar
- **Pros**: No inventory, low upfront cost, quick turnaround
- **Cons**: Higher per-unit cost, less control
- **Cost**: ~$15-25 per deck at low volume

#### Option B: Bulk Manufacturing
- **Provider**: Alibaba manufacturers, local printer
- **Pros**: Lower per-unit cost, higher quality options
- **Cons**: Inventory risk, higher MOQ, longer lead time
- **Cost**: ~$3-8 per deck at 500+ volume

### Recommended Approach
Start with print-on-demand for initial launch and testing, then move to bulk once demand is validated.
